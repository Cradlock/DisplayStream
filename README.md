# Стрим экрана в реальное время 


Мини проект который будет захватывать экран устройства и показывать его в;
---
http://localhost:1234
---
Также включает способ управления курсором мыши:



## Стеки
* Rust: `Основной язык для backend и захвата экрана` 
* HTML: `Веб интерфейс`
* JavaScript: `Логика захвата видео на клиенте`
